# Run_Locally

Clone the project

```bash
https://github.com/saddamarbaa/zoom-clone-app-reactnative-typescript
```

Go to the project directory

```bash
 zoom-clone-app-reactnative-typescript
```

Install dependencies

```bash
yarn install
```

Start the server

```bash
yarn start
```
